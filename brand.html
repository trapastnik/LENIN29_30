22 · R&D · live-DOM на 3D

HTML-in-Canvas — газеты, афиши, документы как тактильные объекты

Технология: живой DOM с CSS/JS захватывается в текстуру и натягивается на 3D-меш Three.js. Через нативное Chromium API ctx.drawElement() (флаг chrome://flags/#canvas-draw-element) обновляется покадрово; в продакшене сейчас html2canvas-fallback с захватом раз в 1.5 сек. Базируется на R&D-брифе XR LAB · 23.04.2026 (cloth / scroll / wall / book / orbit / game).

Газета на тканиPlaneGeometry + волновая деформация
Советская «Правда» как мягкое полотно — для витрин раздела «Поток документов» (Direction C).
Афиша-свитокCylinderGeometry, частичный угол
Агитплакат намотан на цилиндр и медленно вращается — для зала «Гражданская война / агитпроп».
Стеклянная стенаPlaneGeometry + reflection mesh
Карточка партии как стеклянная плашка с зеркальным отражением — для архивного зала.
Книжная страницаPlaneGeometry + изгиб по кривой
Дневник / письмо с лёгким изгибом страницы — для блока биографий и личных документов.

Сцены лениво инициализируются при попадании в viewport. Текстура 1024×1024, anisotropy 8, захват каждые 1.5 секунды (компромисс CPU vs свежесть). Источник прячется на position: absolute; left: -99999px — остаётся в layout, но не виден.
